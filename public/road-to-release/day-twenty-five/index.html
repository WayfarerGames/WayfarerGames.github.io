20 Jun 2023 Polyfury development log

The building work on the house continues, so there’s no major updates from me today. I’m still cracking on with the Triangle Twins, but so far this week I’ve averaged a little over 30 minutes a day 😂

I’ve been sorting out the dynamics of switching between different heptagram/triangle forms. While I’m developing a boss, I have a “debug mode” where it waits after doing an attack, and I can directly select which attack it does next. This is incredibly useful when I’m testing things like how specific patterns interact with each other, but it also means I can boot the game up into a specific boss and test a specific pattern immediately.

What’s next?

Tomorrow I’ve got a good 4 hours to spend on it, so I should have a few more visual things to show for you. I’ve also got to get a new public demo out on the xbox, with a splash screen advertising a competition I’ll be running alongside the hexagon boss.
