14 Jun 2023 Polyfury development log

We’re 20% of the way through! That’s a whole fifth. The game has actually come so far in that time, I’m really pleased with progress. It’s probably time to take stock of what’s left.

There are four levels to do, which feels like a lot given how long the previous three have taken (two years…). I think I’m probably going to have to pick up the pace there a little bit, no? 😂 I’m already a good chunk of the way through the triangle twins, though - there’ll be more to see there in the next few weeks.

I’ve also got to write a little achievements handler and integrate Steam/Xbox social stuff. I’ve already got a skeleton in place, and everything is event-driven so it’s super flexible for things like that. Overengineered code coming in to the rescue!

I also need a couple of extra game modes - the current level select works for the demo, but the main way to play through the game should be consecutive levels, and they need to be locked in the level select menu until you’ve completed them. I’m also going to do a 1CC mode that is a full run through of the game with the same amount of lives and a score that continues between levels. I don’t really think this will take too long since again, everything is incredibly flexible - in fact, it’s in my plans for the next server competition with prizes. Join the mailing list at the bottom of the page to be the first to know about upcoming competitions.

What’s next?

For the next week, I’m going to be concentrating on getting an test of the triangle twins out to the alpha testers. There are still a few patterns left to sort, and some general balance tweaks that need to be made before then, but I’m pretty confident I can get there.
