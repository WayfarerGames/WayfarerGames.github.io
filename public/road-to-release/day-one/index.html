24 May 2023 Polyfury development log

Hey everyone, I'm excited to share with you an update on Polyfury! I've been working on it for way too long, and I'm finally starting to see the light at the end of the tunnel.

Today

I'm all set up for 100 days of dev! I wrote down my to-do list and made sure everything was up on the repository. (Oops, it had been 6 months since my last commit! Don't do what I do. 🙃)

I also set up the blog on Squarespace and created a mailing list.

Oh, and I should probably get to work on the game! The first major bit of work for me to do is a reworking of the options menu to prepare for the final demo update. I'm adding some extra accessibility options and cheat codes, and I'm also cleaning up the code and fixing some controller bugs. I spent around 4 hours on all of this, here’s the changelog:

  • Deleted the old menu, because it was bad and broken. Making progress sometimes requires deleting a bunch of stuff 👍

  • Added a new, better menu copied over from another project, with fancy tabbed navigation that works on a controller

  • Fixed an issue where clicking somewhere with the mouse makes it impossible to navigate the menu

  • Fixed a visual issue with cheat code entry

By the weekend, I hope to be finished with the new menu bits. I need to add a ton of extra accessibility options that are already technically in the game, just not actually exposed anywhere…
